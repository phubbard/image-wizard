About

image-wizard

A local-first, on-device photo indexer for large personal libraries. Walks your filesystem, records EXIF metadata, runs YOLO for object detection, InsightFace for face detection and clustering, and OpenCLIP for free-text semantic search — all on your own machine, with no cloud calls. Everything is stored in a single SQLite database (vector search included, via sqlite-vec). Browse, search, and label from the built-in web UI.

Source: github.com/phubbard/image-wizard
